# moodle-qbank_kia_generator



## Name
KIA KI Question Generator Plugin for Moodle

## Description
Coming soon! This plugin is designed to work with the KIA Connector plugin, allowing you to generate questions for Moodle's question bank using the KIA platform.

## Installation
This plugin needs the KIA Connector plugin to work correctly on your Moodle instance. You can find it [here](https://git.mylab.th-luebeck.de/kia/teaching/moodle-local_kia_connector).

To install the KIA Question Generator plugin, follow these steps:

Navigate to /path/to/moodle/question/bank/ and copy files to a new directory kia_generator or use:
```
git clone https://git.mylab.th-luebeck.de/kia/teaching/moodle-qbank_kia_generator.git kia_generator
```
Then, navigate to your Moodle instance and log in as an administrator. Go to Site administration > Plugins > Install plugins. Moodle will automatically detect the new plugin and prompt you to install it. Follow the on-screen instructions to complete the installation.

## Usage
To use the KIA Question Generator plugin, navigate to the course where you want to add questions. Click on more to find an option to generate questions using the KIA platform. You will be able to select the type of questions you want to generate and configure the settings as needed.
